Сафари за вампиром. Из серии „космические войны“, часть 2.

В 21-м веке лучшим лакомством считается жаркое из вампиров, живущих в соседней галактике.
Вам захотелось хорошо покушать и вы отправились на сафари.

Весьма быстрый псевдотрехмерный космический шутер.

Игра распространялась кооперативом «COMAN».
