Игра, навеянная Moonbugs фирмы Windmill Software.

Изначально эта игра распространялась "Центром Компьютер" в особом формате для защиты от копирования.
Сначала загружался загрузчик, потом заставка (первый скриншот), а потом сама игра.
Большинство взломанных версий этой игры были без заставки.
