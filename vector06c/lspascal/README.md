Язык Паскаль для базовой кассетной конфигурации. Комплект включает в себя текстовый редактор, порождающий p-код транслятор и, в отдельном загружаемом модуле, компилятор.

# Состав архива
- lspas-i.rom — интерпретатор
- lspas-k.rom — компилятор
- pasdoc1.rom — 1 часть описания (первый вариант)
- pasdoc2.rom — 2 часть описания (первый вариант)
- pasdoc3.rom — 3 часть описания (первый вариант)
- pasdoc4.rom — 4 часть описания (первый вариант)
- lspas-d2.rom — вторая часть описания (второй вариант, неполный)
- paslib.m10 — библиотека, загружается по адресам 0100-27FF
- test1pas.m28 — тест, загружается по адресам 2800-34CB

Компилятор языка ЛС-Паскаль.djvu — описание ОС-Паскаль в формате DjVu
