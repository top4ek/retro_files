Эта игра, не совсем игра, а как бы тест на ваше терпение.
Колопанк должен выбраться из подземелья.

Для этого он должен сделать:
1. пройти вправо 100 комнат
2. убить 100 колопанкочудовищ
3. добраться до выхода
4. все!!!

В конце сюрприз!!!
