Патченный Загрузчик «Вектор-06Ц.02» (базовый, 2048 байт, Кишинев).
На экран выводится контактный номер некоего Игоря.
При нажатии F1+F3 выводятся эти же данные.
Больше загрузчик ничем от стандартного не отличается.

Загрузчик для картотеки любезно предоставили dbk, BYTEMAN и Prusak.
