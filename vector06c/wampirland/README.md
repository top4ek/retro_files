Маленький гном попал в страну вампиров.
Ваша задача помочь ему найти выход из нее.
На пути будет встречаться множество опасностей, но волшебные предметы помогут ему справиться с ними.
Эти предметы помогут избавиться от всякой нечисти, кроме летучей мыши и привидения.
Без кола лучше не приближаться к потрошителю, иначе вы погибнете.
