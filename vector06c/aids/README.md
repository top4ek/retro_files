Игра про популярный в 1990-х вирус иммунодефицита человека.
По сути это Minesweeper, в котором сапёр неизвестно чем открывает фоновую картинку.

Есть три версии игры, отличающихся картинками. Два детских СПИДа и один СПИД для взрослых.
