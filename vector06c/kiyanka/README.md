Ваш напоминающий супермарио герой вооружен киянкой, которой он сражается с разными монстрами, которые похитили и охраняют вашу напоминающую суперпринцессупич невесту.

Выпущено две игры, Киянка (зеленая) и Киянка-2 (голубая).
